# Micro-CRM
Spring Boot webservice for contact management of a CRM system.

## Features
* MongoDB integration
* REST API
* Swagger API documentation
* Dockerfile

## Prerequisites
* JDK 11
* Maven

## Installation
* mvn install

## Usage
* Run `./mvnw spring-boot:run`
* REST API: http://localhost:8080/
* Swagger: http://localhost:8080/swagger-ui/
